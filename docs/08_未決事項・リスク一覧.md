## 2. 公式仕様と現行マスタの差

解決済み（2026-07-13）。ユーザー確認とExcelどおり、上段4（ラバムスキル）＋下段13（通常スキル）＝17スキルを現行仕様として確定する。

2026年7月7日のGlobal公式パッチノートにあった「通常スキル14＋Rabamスキル4」という記載は、まな確認により誤り（またはこのプロジェクトの数え方と無関係）と判断し、採用しない。

対応方針:

- MVPの表示は承認済みExcelどおり17スキル（上段4＝ラバムスキル、下段13＝通常スキル）。
- プログラムへ17を固定しない。
- マスタ変更で18以上へ増やせる。
- スキル数が変わっても古い編成を削除しない。

## 3. 主なリスク

### R-001 マスタの陳腐化

- 影響: 誤った装着候補や古いスキルを表示する。
- 対策: `master_version`、更新日表示、公開前検証、旧データ警告。

### R-002 画像とマスタの不一致

- 影響: 別スキルの画像表示、画像欠落。
- 対策: 自動対応チェック、画像なしフォールバック、公開前レポート。

### R-003 ローカルデータ消失

- 影響: ブラウザデータ削除で所持品・編成が消える。
- 対策: Excelバックアップの案内、最終出力日時の表示、定期的な注意。

### R-004 Excel統合の誤操作

- 影響: 所持数の二重加算、編成上書き。
- 対策: 置換を推奨、統合前プレビュー、自動退避、競合確認。

### R-005 大量画像による初期表示遅延

- 影響: 画面表示が遅い。
- 対策: 選択クラス分だけ読込、遅延読込、公開用変換。

### R-006 ファンコンテンツ規約

- 影響: 公開停止や画像差し替えが必要になる。
- 対策: 無料公開、非公式表記、ロゴ非改変、最新ガイド確認。不明点は公式窓口へ確認。

### R-007 数値仕様が未完成 → 概ね解決

- 状況: 深淵・太古・混沌の効果数値が判明（`11_効果マスタ数値.md`）。マスタ既定値＋手編集で対応。系列タイプは深淵のみ存在（太古・混沌なし）と2026-07-13に確定。
- 残課題: 2値（value_a/value_b）の意味ラベル、深淵にダッシュがある効果の等級制限、系列タイプの効果名。
- 対策: 判明分はマスタ化、未確認分は要確認として保持。手編集は引き続き許容。


### R-008 公開基盤・サービス変更

- 影響: Cloudflare PagesやGitHub Pagesの仕様変更により、ビルド・公開手順の修正が必要になる。
- 対策: アプリ本体を特定サービスへ強く依存させず、静的ファイルとして別のホスティングへ移行できる構成を維持する。
